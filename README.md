## 1 - INSTALACIÓN DE DEPENDENCIAS
- Al clonar el repositorio antes de intentar ejecutar el proyecto
se debe instalar las dependencias necesarias, solo se debe ejecutar
"npm install" para que se instalen todas en automático

## 2- VARIBALES DE ENTORNO
- El servidor solo se iniciara si se tiene llenas las variables
de entorno en el archiv .env

## 3- INICIAR PROYECTO
- Iniciar proyecto desde la terminal con "npm run dev"

## ACLARACION
- El proyecto esta desarrollandose bajo la logica de EMS 6 y no common JS
(USO DE IMPORT Y NO REQUIRE)
