## 1 - INSTALACIÓN DE DEPENDENCIAS
- Al clonar el repositorio antes de intentar ejecutar el proyecto
se debe instalar las dependencias necesarias, solo se debe ejecutar
"npm install" para que se instalen todas en automático

## 2- VARIBALES DE ENTORNO
- El servidor solo se iniciara si se tiene llenas las variables
de entorno en el archiv .env (se debe cambiar el nombre del archivo ".env.example" a ".env")

## 3- INICIAR PROYECTO
- Iniciar proyecto desde la terminal con "npm run dev" / al iniciar el proyecto
se mantendra en ejecución todo el tiempo debido a la libreria "nodemon" para
eliminar la necesidad de ejecutar el proyecto por cada cambio realizado, 
cada vez que se haga un cambio se aplicara de inmediato en el proyecto ejecutado en tiempo real

## ACLARACION
- El proyecto esta desarrollandose bajo la logica de EMS 6 y no common JS
(USO DE IMPORT Y NO REQUIRE)
